# Spam-Message-Classifier-using-TF-IDF-and-Multinomial-Naive-Bayes-NLP-Project-
SMS Spam Message Classifier built using Python and Natural Language Processing (NLP). Uses TF-IDF vectorization and Multinomial Naive Bayes to classify messages as Spam or Ham. Achieves ~96% accuracy and demonstrates end-to-end machine learning workflow on real-world SMS data.
